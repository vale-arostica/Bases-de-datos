La edición de un usmito incluye sólo el texto, no se pueden cambiar los tags.

La RRSS creada no pide contraseña para el inicio de sesión de un usuario, dado que no fue especificado en la tarea que era necesario, ni tampoco incluí "password" ni "contraseña" como atributo de usuario en el informe, simplemente se debe ingresar con el usuario, para simplificar la tarea; al ingresar debe escribir @ seguido de su nombre de usuario.

Cuando se registra un usuario, el nombre de usuario no debe llevar caracteres especiales, pues el mismo sistema agrega un @ al username al ingresarlo a la base de datos.

Los tags deben agregarse con # para llevar el orden en la base de datos y para poder identificar que la palabra se trata de un tag en la búsqueda.

Para volver al home sólo es necesario hacer click en la casita de la barra de menú superior.

Evitar "volver atrás" con el navegador, sobre todo en páginas como "search" porque estas páginas dependen del URL enviado, por eso, se debe navegar con los botones en la página, usando la barra de búsqueda, con el menú izquierdo o presionando "home" en la barra superior.
